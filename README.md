# Tiny Guide to Pour-Over Coffee

A **pour-over** is a simple way to brew clean, flavorful coffee using just hot water and ground beans.

## Why people like it
- Easy to learn
- Cheap equipment
- Great flavor clarity

## What you need
1. Dripper
2. Paper filter
3. Fresh coffee
4. Hot water
5. Mug or server

## Basic recipe

> A good starting ratio is **1:16**  
> Example: `20g coffee` → `320g water`

### Steps
- Rinse the filter
- Add ground coffee
- Pour a little water first *(this is called the bloom)*
- Slowly pour the rest in circles
- Let it drain

## Example checklist

- [x] Boil water
- [x] Grind coffee
- [ ] Brew
- [ ] Drink before it gets cold
